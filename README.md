# PictureQuiz


### 랜덤한 사진속 인물이나 캐릭터 퀴즈를 맞추는 웹페이지입니다.

### Deploy : [https://silku.github.io/tangoo9](https://silku.github.io/tangoo9)  

<br>

## 페이지 상세


**Home**
- 로그인/회원가입, 소셜로그인을 할 수 있습니다.

**Quiz**
- 랜덤으로 나오는 퀴즈 게임을 진행합니다.


**Ranking**
- 랭킹을 확인할 수 있습니다.
- 스코어순, 시간순으로 정렬됩니다.
   

**AddQuiz**
- 로그인시 : 퀴즈 출제 및 수정, 삭제

<br>
